# Testing NGRX with Jest Demo Application

This repository is used to distribute the demo App and presentation I created for the Angular Munich Meetup #24 on February 14th 2019.

## Setup

- clone repository
- run `npm install`

## Running the App

- run `ng serve`

## Running the tests

- single run: `ng test`
- watch mode: `ng test --watch`
